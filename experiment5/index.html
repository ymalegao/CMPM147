Description

This is a living Impressions generator that uses mutations to get to the best evaluted similarity score. YOU HAVE TO RELOAD SOMETIMES AFTER SWITCHING THE IMAGE, if it doesnt get the mutation in like 20-30 seconds please consider reloading

Technical

  1. You are able to choose from 5 options of what photo you want, and in seconds the generator is able to render shapes that match the color of the spot where it renders the shapes. This is done by mapping the X and Y coordinate that is chosen for the shape, to the image, and then getting the color of the image on that spot. Once the first set of shapes are rendered, the mutations will continue to generate shapes that are closer and closer to the image that we started with, using a evolutionary algorithm that keeps a higher score
